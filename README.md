# Project Landing Page

This is a fully styled landing page built from scratch as part of The Odin Project curriculum. The goal of this project is to practice transforming a design mockup into a structured, responsive web page using semantic HTML and advanced Flexbox layouts.

## 🚀 Live Demo
You can view the live deployment of this project here: [Insert your GitHub Pages link here]

## 🛠️ Technologies Used
* **HTML5** - Semantic structure
* **CSS3** - Custom layouts, positioning, and variables
* **Flexbox** - Component and page-level alignment

## 🧠 Key Learning Objectives
* Structuring complex layouts by nesting flexible boxes.
* Implementing typography and color design guidelines precisely from a visual spec sheet.
* Managing clean alignment across varying screen sections (Hero, Info Grid, Testimonials, Call to Action).
